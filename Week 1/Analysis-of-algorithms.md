# Analysis of Algorithms

## key Idea

A single problem can have multiple solutions, and each solution may have a different time complexity.

<hr />

## Example: Sum of first n natural numbers

<b>Input:</b> n=3

<b>Output:</b> 6

### Solution 1

n\*(n+1)/2
Time Complexity: O(1)

<hr />

### Solution 2

1+2+3
Time Complexity: O(n)

<hr />

### Solution 3

Repeated Addition
Time Complexity: Higher then O(n)
